# Bhavans Android Student Portal

A simple Android student portal app for Bhavans College.

## Features

- Welcome page with Start button
- Dashboard with college logo and six subject boxes
- Subject sections:
  - Syllabus
  - Notes
  - Important Questions
  - Upload Homework / Notes
- College information sections:
  - Time Table with image
  - Map of College with image
  - Floors and room numbers
  - All Streams / Courses

## Tech Stack

- Java
- XML Layouts
- Android SDK
- Gradle

## Folder Structure

```text
app/src/main/java/com/example/bhavans/
app/src/main/res/layout/
app/src/main/res/drawable/
```

## Build Command

```bash
./gradlew assembleDebug
```

APK location:

```text
app/build/outputs/apk/debug/app-debug.apk
```
